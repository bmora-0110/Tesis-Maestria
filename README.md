# 📘 Tesis de Maestría  
Transferencia de Conocimiento Universidad–Regiones en Colombia

## 📌 Descripción
Este repositorio contiene el código y los datos utilizados para analizar la dinámica de transferencia de conocimiento desde las universidades hacia los Sistemas Regionales de Innovación en Colombia.

Se utilizan modelos:
- ARMAX (Identificación de Sistemas)
- VARX (Econometría multivariada)

---

## 📂 Archivos

### 00_limpieza_graficos.R
Limpieza, estandarización de datos y gráficos descriptivos.

### 01_armax.R
- Modelo ARMAX (2,2)
- Ecuaciones en diferencias
- Validación de supuestos

### 02_varx_triado.R
- Modelo VARX (1)
- Matrices de influencia
- Funciones impulso-respuesta
- Validación de supuestos

### 03_mapas.R
Generación de mapas de regiones y departamentos.

### base_datos.xlsx
Base de datos del estudio.

---

## ⚙️ Requisitos
- R
- Librerías: vars, tseries, forecast, ggplot2, readxl

---

## 📊 Enfoque
- ARMAX → dinámica temporal
- VARX → interacciones entre regiones
